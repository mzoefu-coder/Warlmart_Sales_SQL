# Warlmart_Sales_SQL

The main purpose of this project is to gain an understanding of Walmart's sales data and explore the factors that influence sales across different branches.
# Objectives
- Analyze the data to learn more about product lines, identify the best-performing ones, and pinpoint areas where other product lines need improvement.

- Identify different consumer categories, comprehend purchasing patterns, and assess the profitability of each of these customer segments.
